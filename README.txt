Content Downloader
Content Downloader — это приложение для получения URL по ключевым словам, загрузки их содержимого и сохранения в локальном хранилище браузера для оффлайн-доступа.

Как запустить проект локально
Шаг 1. Клонирование репозитория
Склонируйте проект на локальный компьютер:

git clone https://github.com/Retr0Gamer01/content-downloader.git
cd content-downloader

Шаг 2. Установка зависимостей
Убедитесь, что у вас установлен Node.js (версия 16 или выше). Установите зависимости:

npm install

Шаг 3. Запуск сервера
Для запуска сервера выполните:

node server.js
По умолчанию сервер будет доступен по адресу http://localhost:10000.

Шаг 4. Запуск клиентской части
Откройте файл index.html в любом современном браузере.
Убедитесь, что сервер работает, чтобы взаимодействие с ним было доступным.

Публичный доступ
Этот проект доступен публично, поэтому вам не нужно разворачивать его локально. Вы можете использовать его, перейдя по следующим ссылкам:

Клиентская часть: https://Retr0Gamer01.github.io/content-downloader/
Серверная часть: Развернута на Render,https://keyword-server-xgg3.onrender.com.
Убедитесь, что клиентская и серверная части связаны корректно для работы.

Использование
Перейдите по ссылке на клиентскую часть.
Введите ключевое слово в поле на странице.
Нажмите кнопку "Получить список URL".
Выберите URL и начните загрузку.
Следите за прогрессом загрузки.
Просмотрите загруженный контент.
